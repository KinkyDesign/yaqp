Upload a file on the server

A valid pair of username and password are required to allow you to upload an arff file on the server. The size of the file you want to upload should not exceed 1MB. Use the following form to upload your file. If the upload is successful, a URI will be assigned to your file. Keep that in mind to be able to use this dataset to train or evaluate a model.
